# FOTO-NET Examples

## `predict_image.py`

Run inference on one image.

## `predict_folder.py`

Run inference on a folder of images.

## `train_custom_yolo.py`

Train on a YOLO-format dataset.

## `export_onnx.py`

Export a checkpoint or model scale to ONNX.

## `transform_crop.py`

Use the transform API to expand, move, crop, and test a detection region.
